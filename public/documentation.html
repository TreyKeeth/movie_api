Web application that provides information about various movies, directors, and genres. Users can subscribe and create lists such as a list of favorite movies.


Purpose URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Add a user /users POST None A JSON object holding data about the user to add, structured like: {"Username": "",} A JSON object holding data about the user that was added, including an ID: {"Username": "", "id": ""}
Update a user (username) /users/[id] PUT :id None A JSON object holding data about the user that was updated, including an ID: {"Username": "", "id": ""}
Remove a user /users/[id] DELETE :id None Text message indicating that the user has been deleted
Add a movie to user favorites /users/[id]/[movieName] POST :id, :movieName None Text indicating that the movie has been added to the user's list of favorite movies
Remove a movie from user favorites /users/[id]/[movieName] DELETE :id, :movieName None Text indicating that the movie has been removed from the user's list of favorite movies
Get a list of movies /movies GET None None A JSON object holding data about all the movies
Get data about a single movie by movie name /movies/[movieName] GET :movieName None A JSON object holding data about a single movie, containing the description, genre, director, image, URL, and whether it's featured or not
Get data about a genre by name /movies/[genres]/[genreName] GET :genres, :genreName None A JSON object holding data about a single genre
Returns data about a director by name /movies/[directors]/[directorName] GET :directors, :directorName None A JSON object holding data about a single director, containing the director's bio, birth year, and death year
